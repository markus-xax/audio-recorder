# System Audio Recorder

Приложение-диктофон для Windows и macOS, которое записывает системный звук (звук из браузера, приложений и т.д.), а не звук с микрофона.

## Возможности

- ✅ Запись системного звука (браузер, приложения)
- ✅ Не использует микрофон устройства
- ✅ Простой и понятный интерфейс
- ✅ Кроссплатформенность (Windows и macOS)
- ✅ Сохранение в формате MP3

## Требования

### Для разработки:
- Node.js (версия 14 или выше)

### Для пользователей:
- **Ничего дополнительно устанавливать не нужно!** FFmpeg встроен в приложение.

### Дополнительно для macOS (опционально):
Для лучшей работы рекомендуется установить **BlackHole** - виртуальный аудио драйвер:
1. Скачайте с https://github.com/ExistentialAudio/BlackHole
2. Установите драйвер
3. В настройках системы выберите BlackHole как устройство вывода (опционально)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

## Сборка для распространения

### Windows:
```bash
npm run build:win
```

### macOS:
```bash
npm run build:mac
```

Собранные приложения будут в папке `dist/`.

## Использование

1. Запустите приложение
2. Нажмите "Начать запись"
3. Выберите место для сохранения файла
4. Запись начнется автоматически
5. Нажмите "Остановить" для завершения записи

## Важные примечания

### Windows:
- Приложение использует WASAPI loopback для захвата системного звука
- FFmpeg встроен в приложение, дополнительная установка не требуется
- Если запись не работает, проверьте настройки аудио устройства
- Альтернатива: установите VB-Audio Virtual Cable и используйте его как устройство вывода

### macOS:
- По умолчанию используется системный звук через Core Audio
- Для более надежной работы рекомендуется установить BlackHole
- Если звук не записывается, проверьте настройки приватности в Системных настройках

## Технические детали

- **Платформа:** Electron
- **Формат записи:** MP3 (192 kbps, 44.1 kHz, стерео)
- **Захват аудио:** 
  - Windows: WASAPI loopback
  - macOS: AVFoundation (Core Audio)

## Устранение неполадок

### FFmpeg не найден
- Это не должно происходить в собранном приложении, так как FFmpeg встроен
- Если ошибка возникает, переустановите приложение
- В режиме разработки убедитесь, что установлены все зависимости: `npm install`

### Звук не записывается
- **Windows:** Проверьте, что устройство вывода активно
- **macOS:** Установите BlackHole и выберите его как устройство вывода, или проверьте настройки приватности

### Ошибки при записи
- Убедитесь, что у приложения есть права на запись файлов
- Проверьте, что выбранная папка существует и доступна для записи
